# Factory Safety Game

An interactive browser-based factory safety training game designed to make manufacturing onboarding more engaging through hazard recognition, scoring, and certification.

## Features
- Interactive safety scenarios
- Hazard recognition
- Score tracking
- Certification quiz
- Responsive design

## Tech Stack
- React
- Vite
- JavaScript
- HTML
- CSS

## Why I Built It
I created this project after seeing traditional safety training rely heavily on forms and quizzes. My goal was to build a more interactive learning experience for manufacturing employees.
