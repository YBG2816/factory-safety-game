# Factory Safety Game

An interactive browser-based safety training game built to modernize manufacturing onboarding through scenario-based learning, hazard recognition, scoring, and certification.

## The Problem

While working in manufacturing, I noticed that much of the safety training relied on forms, presentations, and basic quizzes. Although the information was important, the learning experience wasn't very engaging.

I wanted to explore whether software and gamification could make safety training more interactive, memorable, and enjoyable while still teaching the same core concepts.

## Features

- Interactive safety scenarios
- Hazard recognition challenges
- Score tracking
- Certification quiz
- Responsive design
- Instant feedback
- Browser-based deployment

## Why I Built It

I built this project after seeing an opportunity to improve the way manufacturing safety training is delivered.

Instead of relying only on static forms and quizzes, I wanted to create an experience where employees actively identify hazards, make decisions, and learn through interaction.

This project combines my manufacturing experience with software development to demonstrate how technology can improve training, engagement, and knowledge retention.
